Универсальный скрипт на **Python** для скачивания аудиоконтента с
**YouTube** и **YouTube Music**.\
Программа ориентирована на коллекционеров музыки: она автоматизирует
создание папок, конвертацию аудио и заполнение тегов.

# 🛠 1. Пошаговая установка

Для работы программы на **macOS** выполните следующие шаги.

## Шаг 1: Установка Python

Убедитесь, что у вас установлен Python **3.10 или новее**.

``` bash
python3 --version
```

## Шаг 2: Установка кодеков (FFmpeg)

Это критически важный компонент для конвертации видео в MP3.

Установка через **Homebrew**:

``` bash
brew install ffmpeg
```

## Шаг 3: Установка библиотек

Основной движок загрузки --- **yt-dlp**.

``` bash
pip install yt-dlp
```

# ⚙️ 2. Настройка параметров в `main.py`

В файле `main.py` находятся ключевые переменные, которые определяют
поведение скрипта.

### USE_FILE_BY_DEFAULT

Определяет режим работы программы.

-   `True` --- скрипт читает ссылки из файла `links.txt`
-   `False` --- программа запрашивает ссылку напрямую в консоли

### DOWNLOAD_DIR

Папка, в которую будет сохраняться вся скачанная музыка.

Пример:

    ~/Downloads/YouTubeMusic

### LINKS_FILENAME

Имя файла со списком ссылок для скачивания.

По умолчанию:

    links.txt

# 🚀 3. Полный функционал программы

Скрипт выполняет полный цикл обработки.

## ✅ Валидация ссылок

Программа игнорирует любой мусор в файле `links.txt`.\
Если строка не начинается с `http`, она будет автоматически пропущена.

## 📁 Умное распределение по папкам

### Режим альбома

Если ссылка ведет на **плейлист или альбом**, создается папка с чистым
названием альбома.

Приписка **"Album -"** автоматически удаляется.

### Режим сингла

Если ссылка ведет на **одно видео**, трек сохраняется в папку:

    All songs

## 🎧 Автоматическая конвертация

-   скачивание аудио в максимальном качестве
-   перекодирование в **MP3 (192 kbps)**

## 🏷 Метаданные и обложки

Скрипт автоматически:

-   записывает **название трека**
-   записывает **исполнителя**
-   добавляет **ID3-теги**
-   вшивает **обложку высокого разрешения**
-   сохраняет все обложки в отдельную папку **Covers**

## 📂 Авто‑открытие папки

После завершения работы скрипт автоматически откроет папку с результатом
в **Finder**.

# 💡 Использование

Программа поддерживает два режима работы.

## Вариант A --- Массовая загрузка

1.  В `main.py` установите:

```{=html}
<!-- -->
```
    USE_FILE_BY_DEFAULT = True

2.  Добавьте ссылки в файл:

```{=html}
<!-- -->
```
    links.txt

3.  Запустите программу:

``` bash
python3 main.py
```

## Вариант B --- Быстрая загрузка одной ссылки

1.  Установите:

```{=html}
<!-- -->
```
    USE_FILE_BY_DEFAULT = False

2.  Запустите программу:

``` bash
python3 main.py
```

3.  Вставьте ссылку в консоль, когда программа её запросит.
