# 🎵 YouTube Music Downloader (Artist & Album Edition)

Универсальный скрипт на Python для скачивания аудиоконтента с YouTube и
YouTube Music. Программа ориентирована на коллекционеров музыки: она
берет на себя всю рутину по созданию папок, конвертации и заполнению
тегов.

------------------------------------------------------------------------

## 🛠 1. Пошаговая установка

Для работы программы на **macOS** выполните следующие шаги:

### Шаг 1: Установка Python

Убедитесь, что у вас установлен Python 3.10 или новее. Проверить можно
командой в терминале:

``` bash
python3 --version
```

### Шаг 2: Установка кодеков (FFmpeg)

Это критически важный компонент для конвертации видео в MP3.

**На Mac (через Homebrew):**

``` bash
brew install ffmpeg
```

### Шаг 3: Установка библиотек

Основной движок загрузки --- **yt-dlp**. Установите его командой:

``` bash
pip install yt-dlp
```

------------------------------------------------------------------------

## ⚙️ 2. Настройка параметров в `main.py`

В файле `main.py` находятся ключевые переменные, которые определяют
поведение скрипта:

  ----------------------------------------------------------------------------
  Параметр                Описание                Значение
  ----------------------- ----------------------- ----------------------------
  USE_FILE_BY_DEFAULT     Режим работы. **True**  True / False
                          --- читает ссылки из    
                          файла. **False** ---    
                          запрашивает ссылку в    
                          консоли.                

  DOWNLOAD_DIR            Путь сохранения. Папка, `~/Downloads/YouTubeMusic`
                          куда попадет музыка.    

  LINKS_FILENAME          Имя файла. Текстовый    `links.txt`
                          файл со списком ссылок. 
  ----------------------------------------------------------------------------

------------------------------------------------------------------------

## 🚀 3. Полный функционал программы

Скрипт выполняет полный цикл обработки:

### ✅ Валидация ссылок

Программа игнорирует любой мусор в файле `links.txt`. Если строка не
начинается с `http`, она будет пропущена.

### 📁 Умное распределение по папкам

**Режим альбома:**\
Если по ссылке несколько треков (плейлист/альбом), создается папка с
чистым названием альбома (без приписки *"Album -"*).

**Режим сингла:**\
Если ссылка ведет на одиночное видео, трек сохраняется в общую папку
**All songs**.

### 🎧 Автоматическая конвертация

Загружает аудио в максимальном качестве и перекодирует в **MP3 (192
kbps)**.

### 🏷 Метаданные и обложки

-   Вшивает название и исполнителя в **ID3‑теги**
-   Вклеивает **обложку высокого разрешения** прямо в MP3‑файл
-   Дополнительно копирует все обложки в отдельную папку **Covers**

### 📂 Авто‑открытие

После завершения работы скрипт автоматически откроет папку в **Finder**.

## 💡 Использование

Программа поддерживает два способа работы:

### Вариант A: Массовая загрузка (через файл)

1.  Установите в `main.py` значение `USE_FILE_BY_DEFAULT = True`
2.  Добавьте ссылки в файл `links.txt`
3.  Запустите:

``` bash
python3 main.py
```

### Вариант B: Быстрая загрузка одной ссылки

1.  Установите в `main.py` значение `USE_FILE_BY_DEFAULT = False`\
    (или просто оставьте файл `links.txt` пустым)
2.  Запустите:

``` bash
python3 main.py
```

3.  Вставьте ссылку прямо в консоль, когда программа её запросит.
